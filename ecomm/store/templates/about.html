Welcome To Django Ecommerce



Program Women in Tech 2024 ini ditujukan untuk memberikan pengembangan pengetahuan dan kemampuan bagi perempuan di bidang teknologi dan membuka peluang karir yang lebih banyak. Dengan diadakannya program ini, juga dapat membuka jalan bagi perempuan untuk menciptakan inovasi teknologi yang dapat memberikan dampak positif bagi perempuan dan juga masyarakat.

Peserta akan mempelajari dasar pemrograman Python terkait variabel, tipe data yang berbeda, cara melakukan operasi dasar, cara menyimpan dan memanipulasi kumpulan data, membaca dan menulis berkas, serta Django sebagai tools yang digunakan untuk mengerjakan project akhir. Pada akhir pelatihan, peserta akan menyelesaikan project secara berkelompok dengan menggunakan bahasa pemrograman Python dengan tools Django.

Kelompok 7 membuat sebuah ecommerce website sederhana sebagai syarat kelulusan dari program pelatihan ini.
